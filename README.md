# Smart-Workforce-Upskilling_Application
Development of a Smart Workforce Upskilling Framework for Corporate Employees| Python, Flask, Groq API, SQLite, Power BI |
•	Developed an AI-powered corporate upskilling platform that recommends personalized learning paths for employees.
•	Built backend with Flask and Groq API (LLM integration); used SQLite for data storage and Power BI for analytics dashboards.
•	Presented as MBA final year capstone project supervised by faculty at Institute of Business Management, Jadavpur University.
